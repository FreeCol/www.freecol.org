Current progress
Wednesday, 18 August 2004
The next version of FreeCol will be released sometime in September. Check out the Screenshots section for two screenshots from the development version.
 
Background images and illustrations
Friday, 20 February 2004
We have got permission to use graphics from the David Rumsey Map Collection. This collection has over 8,800 maps on-line. Many of these maps contains illustrations we may later use in the game.
 
Status Update
Saturday, 18 January 2003
The FreeCol project is currently in need of graphics for things such as cities (colonies), goods, 16th/17th century artwork (drawings of people for example), icons/buttons, ... If you are a graphical artist then you can be of great help to the project.

We have managed to port the server to Java. As a result, the development version of FreeCol is now entirely written in Java. This version can be found in CVS.
 
2 More Screenshots Added
Thursday, 31 October 2002
Check out the Screenshots section for 2 more screenshots of the Java client.
Ships can now carry units and units will automatically disembark when the user moves a ship into land. The states of the units are now also visible on the map. What you can't see from these screenshots is that all possible forms of scrolling and automatically recentering of the view have been implemented.
